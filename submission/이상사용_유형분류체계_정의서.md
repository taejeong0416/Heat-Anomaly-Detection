# 열 원격검침 이상사용 유형 분류 체계 정의서

## 1. 개요

지역난방 열량계 원격검침 시간별 사용량 데이터(2021~2025, 약 1,665만 행)를 분석하여, 이상사용 패턴을 9개 유형으로 분류하는 체계를 정의한다.

## 2. 분류 체계 구조

```
원격검침 데이터 (설비-일 단위, 24시간 사용량)
       │
       ▼
┌─────────────────────────────┐
│ 1계층: 데이터 품질 이상 (규칙)  │  ← 검침 오류, 결측 등
│   → 전처리 단계에서 제거/보정   │
└──────────┬──────────────────┘
           │ (품질 통과 데이터)
           ▼
┌─────────────────────────────┐
│ 2계층: 이상사용 탐지 (모델+규칙) │  ← 비정상적 열 사용 패턴
│   → 9개 유형으로 분류           │
└─────────────────────────────┘
```

## 3. 이상사용 9유형 정의

### 우선순위

복수 유형에 동시 해당 시 아래 순서로 주 유형을 결정한다.

> 장기미사용후급증 > 계절역행 > 야간이상 > 주말이상 > 급증 > 기저유량이상 > 연속가동 > 간헐사용 > 패턴이탈

### 유형별 상세

| # | 유형명 | 판정 조건 | 실무적 의미 |
|:-:|--------|----------|------------|
| 1 | **장기미사용후급증형** | 설비별 직전 7일 연속 사용량=0 이후 당일 사용량 급증 (ma7_ratio 상위 20%) | 공실 후 입주, 열량계 재작동 |
| 2 | **계절역행형** | 비활성시즌에 총사용량이 종별 상위 분위 초과 (주택 99%ile / 나머지 95%ile) | 열손실, 배관 누수 |
| 3 | **야간이상형** | 야간(22~06시) 사용 비율이 종별·월 정상 평균 대비 유의하게 높음 (주택 +3σ / 나머지 +2σ) | 밸브 미폐쇄, HVAC 스케줄 오류 |
| 4 | **주말이상형** | 업무/공공용 건물이 주말·공휴일에 평일 수준(중위수 80% 이상) 사용 | 무인 건물 냉난방 미해제 |
| 5 | **급증형** | 7일 이동평균 대비 사용량 비율 상위 10% AND 그룹 내 Z-score 상위 10% | 배관 누수, 밸브 고장, 과다계량 |
| 6 | **기저유량이상형** | 시간대별 최소 사용량(baseload)이 종별·시즌 95%분위 이상 | 지속적 미세 누수 |
| 7 | **연속가동형** | 시간대별 변동 극소(hourly_std 하위 5%) AND 기저유량 높음(상위 80%) | 보일러 상시 가동, 24시간 HVAC |
| 8 | **간헐사용형** | 변동계수(cv)와 시간별 표준편차 모두 상위 5% | 열량계 간헐 결함, 비정형 거주 패턴 |
| 9 | **패턴이탈형** | GMM 로그우도 하위 5% AND (AE 복원오차 상위 5% OR MP 이상점수 상위 5%), 또는 상위 8유형 미해당 잔여 | 기타 비전형 사용 패턴 |

### 종별 적용 차이

| 종별 | 적용 유형 | 특이사항 |
|------|----------|----------|
| 주택용 | 9유형 전체 | 야간이상 +3σ (정상 새벽 가동 패턴 회피) |
| 업무용 | 9유형 전체 | 주말이상형 적용, 야간이상 +2σ |
| 공공용 | 9유형 전체 | 주말이상형 적용, 야간이상 +2σ |
| 냉수용 | 9유형 전체 | 시즌 반전 적용 (활성시즌 = 냉방시즌 6~9월) |

## 4. 탐지 알고리즘

### 모델 기반 탐지 (이상 여부 판정)

| 알고리즘 | 역할 | 특징 |
|---------|------|------|
| Isolation Forest | 다차원 피처 공간에서 고립도 기반 이상 탐지 | 종별별 학습, 38개 피처 입력 |
| Autoencoder | 24시간 패턴 복원 오차 기반 탐지 | 24차원 → 8차원 → 24차원, 종별별 학습 |
| Matrix Profile | 설비별 시계열 자기참조 기반 탐지 | 자기 과거 대비 이질적 구간 탐지 |
| 통계 (Z-score, IQR) | 종별·지사·월 그룹 내 극단값 탐지 | Context-aware 정규화 |

### 규칙 기반 분류 (유형 결정)

모델 탐지와 독립적으로, 9유형 규칙을 전체 데이터에 적용하여 유형을 분류한다. 최종 이상 판정은 모델과 규칙의 OR 결합이다.

## 5. 성능 평가 결과

합성 이상 주입(Synthetic Anomaly Injection) 방식으로 7유형 이상을 인위 생성하여 평가.

| 알고리즘 | Precision | Recall | F2 |
|---------|:---------:|:------:|:--:|
| **본 알고리즘** | 0.639 | **0.913** | **0.841** |
| 베이스라인 (전년·전전년 대비 ±30%, 둘 다 가용 AND) | **0.799** | 0.414 | 0.458 |

- Recall 0.913: 이상의 91%를 탐지 (베이스라인 대비 **+49.9%p**)
- 7유형 전체에서 베이스라인 Recall 상회
- 베이스라인이 탐지 불가능한 패턴이탈형(총사용량 변화 없이 시간 패턴만 변화)도 70.6% 탐지
