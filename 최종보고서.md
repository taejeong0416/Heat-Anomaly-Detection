## Phase별 핵심 결과

### Phase 0. 데이터 통합 및 기본 정제

- 60개 월별 CSV를 단일 parquet으로 통합 (16,868,714행)
- 총사용량 = 1~24시 합으로 재계산, `유형` 컬럼 drop (전 기간 `-`)
- 파생 컬럼 추가: 연도, 월, 일, 요일, 계절, 난방시즌, 공휴일
- 설비 종별 변경 0건 확인 (전 기간 일관)
- **이상값은 EDA 후 판단으로 이월**

### Phase 1. EDA

- 종별 간 스케일 격차 확인 → **종별 분리 분석 필수** 확정
- 난방시즌/비난방시즌 패턴 차이 극명, 업무용·공공용 평일/주말 차이 두드러짐
- 음수값 6,604건(0.04%), NaN 포함 209,894건(1.2%), NaN 30%+ 설비 45개
- 지사 익명화로 기온 미사용 결정 → 월/계절로 대체
- **Phase 2 전처리 파라미터 전량 확정**
- `전처리방침.md`, `도메인조사.md` 작성

### Phase 2. 정밀 전처리

- **파이프라인 개편**: PNNL-24331 기반 9단계로 변경 → **실행 완료 (607초)**
- 음수 처리: 1단계 -1,811행, 2단계 Case1=7,028/Case2=3,714/Case3=548, 3단계 192셀
- 결측 처리: 4단계 -122,828행(NaN≥3), 5단계 제거 13,920행/보간 75,564행
- 설비 필터: 6단계 -279설비(1년미만), 7단계 -29설비(결측28%+), 8단계 -1설비(처리비율30%+)
- 극단값: 9단계 행 제거 235건, clip 332,806셀
- 극단값 threshold: 주택 2.59, 업무 2.74, 공공 4.38, 냉수 3.23 Gcal
- 정규화 파라미터: 종별×활성시즌 8그룹 Z-score 산출 완료
- **최종: 16,868,714 → 16,650,282행 (98.7%), 10,126 → 9,814설비**
- **품질 검증 ALL PASS**: 음수 0, NaN 0, 종별 제거율 전부 <5%

### Phase 3. Feature Engineering

- 14개 신규 피처 + 24개 시간대 비율 피처 생성 (220초)
- 기본통계: cv, hourly_std, peak_hour, baseload
- 패턴비율: hour_ratio_1~24, night_ratio, day_ratio
- 추세비교: ma7_ratio, ma30_ratio
- 시계열: autocorr_lag1, hour_diff_max, hour_diff_std, zero_count
- 메타: is_holiday (하드코딩), is_weekend
- 기존 컬럼 활용: 총사용량, 난방시즌, 냉방시즌, 계절, 월, 요일, 종별, 지사
- 중복 피처 정리: daily_mean(=총사용량/24), is_heating_season(=난방시즌), season(=계절) 제거
- NaN 18.9% (cv, night/day_ratio, autocorr) — 비난방시즌 제로 사용량으로 인한 정상 결측
- **최종: 16,650,282행, 78컬럼, 1,231.6MB**

## 이상 유형 분류 체계

> 상세: `이상유형정의.md` 참조

### 1계층: 데이터 이상 (규칙 기반, 우선순위 계층)

모델 전단 필터. 해당 조건에 걸리면 이상유형 라벨을 부여하고 모델에 넣지 않는다.

| 단계 | 유형 | 조건 | 학습 시 |
|:----:|------|------|:-------:|
| 1 | 총량음수 | `sum(24h) < 0` | 행 제거 |
| 2-3 | 음수 처리 | PNNL 인접양수 상쇄 → 무상쇄 NaN | 값 보정 |
| 4 | 데이터수집이상 | `nan_count >= 3` | 행 제거 |
| 5 | 연속결측이상 | `max_consecutive_nan >= 2` | 행 제거 |
| 9 | 극단값이상 | `any(hour > threshold × 100)` | 행 제거 |

### 2계층: 이상사용 (모델 기반)

데이터 이상 필터를 통과한 데이터에 대해 정상 패턴 모델로 탐지. Phase 5~6에서 확정.

| 유형 후보 | 패턴 | 실무적 의미 |
|----------|------|------------|
| 급증형 | 전일 대비 급등 | 배관 누수, 밸브 고장 |
| 야간 이상형 | 업무용 야간/주말 비정상 사용 | 밸브 미폐쇄, HVAC 오류 |
| 장기 미사용 후 급증형 | 연속 제로 → 급증 | 열량계 재작동 |
| 계절 역행형 | 비활성시즌 고사용 (난방: 4~10월, 냉수: 10~5월) | 열손실, 배관 누수 |
| 주말/공휴일 이상형 | 업무용 주말 ≈ 평일 | 무인 건물 HVAC 미해제 |
| 기저 유량 이상형 | 최소 시간대 지속 높음 | 배관 누수 |
| 패턴 이탈형 | 클러스터 중심 거리 상위 | 설비 오작동, 용도 변경 |
