## 핵심 결과

- 60개 월별 CSV(2021-01 ~ 2025-12)를 단일 parquet로 통합: **16,868,714행**
- 컬럼 정리 및 파생 컬럼 9개 추가 (연도, 월, 일, 요일, 계절, 난방시즌, 공휴일 등)
- `유형` 컬럼 전 기간 `-`만 존재 확인 → drop
- 총사용량 = 1~24시 합으로 재계산
- 음수값·극단값·NaN은 **제거하지 않고 보존** (EDA 후 판단으로 이월)

## 처리 상세

| 처리 단계 | 내용 |
|-----------|------|
| 파일 통합 | 60개 `anonymized_YYYYMM.csv.gz` → 단일 DataFrame |
| 인코딩 | `utf-8-sig` |
| 컬럼명 변경 | `총` → `총사용량` |
| 유형 컬럼 | 전 기간 `-`만 존재 → drop |
| 총사용량 재계산 | 1시~24시 합산으로 덮어쓰기 |
| 파생 컬럼 | 연도, 월, 일, 요일, 계절, 난방시즌, 공휴일 |
| 공휴일 | `holidays` 라이브러리 (한국 공휴일) |
